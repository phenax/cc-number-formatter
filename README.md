# Card number Formatter
Formats the card number

## Getting Started


### Installing

Install this library as a dependency in your project

```
yarn add ${x}
```
OR
```
npm i ${x} --save
```

## Running the tests

* Clone this repository to your local machine
* Run `yarn test`
